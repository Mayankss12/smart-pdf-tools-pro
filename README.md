# Mayank Smart PDF Tools Pro

A free-first professional PDF tools website starter.

## Features included

- Homepage
- PDF tools grid
- PDF Editor UI
- Upload/drop PDF area
- Tools: Text, Image, Highlight, Draw, Sign, OCR, Rotate, Delete, Export
- Pricing page
- Dashboard placeholder
- Supabase-ready setup
- Railway backend placeholder

## How to run

```bash
npm install
npm run dev
```

## Deploy

Upload this project to GitHub, then import it into Vercel.
Fresh deployment trigger after compress fix.
